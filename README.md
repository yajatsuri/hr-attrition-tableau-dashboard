# Employee Attrition Analysis Dashboard

## 📊 Overview
Interactive Tableau dashboard analyzing employee attrition 
patterns across 1,470 employees to identify key retention risks.

## 🔍 Key Insights
- **16.1% attrition rate** across the organization
- R&D department has highest attrition (133 employees)
- Employees earning <5K salary show highest attrition (163)
- 30-39 age group is most at-risk (89 employees left)

## 🛠 Tools Used
- Tableau Public
- Dataset: IBM HR Analytics (Kaggle)

## 📌 Dashboard Features
- KPI cards for Total Employees, Attrition Count & Rate
- Attrition breakdown by Department, Job Role, Age, Salary, Education
- Interactive filters by Department
